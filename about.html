📘 About TaskBoard Lite

TaskBoard Lite is a simple, fast, and local-first task management tool built for small teams, students, and freelancers. No accounts, no servers — everything happens right in your browser. ✨

🌟 Key Features
  • ✅ To Do / In Progress / Done workflow
  • 🔄 Drag & drop tasks
  • 💾 Save/Load board as JSON
  • 🌗 Dark mode (persistent)
  • 🚀 Works offline, 100% local
